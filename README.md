# SHL Assessment Recommender AI Bot

An AI-powered conversational agent that recommends SHL talent assessments
based on job roles and hiring requirements. Built with FastAPI and Groq's LLM API.

---

## What It Does

Hiring managers describe a role or requirement in plain English.
The bot asks clarifying questions if needed, then recommends the most
relevant SHL assessments from a structured catalog — with real URLs,
test types, and job level matching.

- Multi-turn conversation (up to 8 turns)
- Refuses off-topic questions and prompt injection attempts
- Every recommendation validated against real catalog data (no hallucinations)
- Returns clean structured JSON responses
